# ProPlan — AHMP

Application de planification de tournées VMC et Éradication.

## Déploiement sur Vercel

1. Importer ce dépôt sur GitHub
2. Connecter à Vercel → New Project → importer le repo GitHub
3. Vercel détecte automatiquement React → cliquer Deploy
4. L'app est disponible sur votre-url.vercel.app

## Structure
- `src/App.js` — application complète ProPlan
- `public/index.html` — page HTML de base
- `package.json` — dépendances React

## Configuration Supabase
Les clés sont déjà intégrées dans src/App.js.
